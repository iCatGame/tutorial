# 扩展图像

“平移（Pan）”选项允许您沿选定方向扩展图像的画布，而不更改原始图像的内容。新扩展的画布将使用提示和原始图像的指导进行填充。

## 使用

[提高图像分辨率](../upscalers/)之后将出现`⬅️` `➡️` `⬆️` `⬇️`四个平移按钮。平移时，仅使用距离图像侧面最近的 512 个像素以及提示词来确定新部分。

![平移按钮](./img/MJ_PanButtons.jpg)

`提示词：A vibrant fantasy landscape`
