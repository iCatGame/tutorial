# 扩大器

扩大器（Upscalers）用于提升图像分辨率。下面介绍如何使用。

## 生成图像

使用`/imagine`命令创建图像。

![](./img/MJ_Upscale_Prompt.gif)

## 放大图像

使用 U 按钮将所选图像与网格分开。

![](./img/MJ_Upscale_ImageGrid.png)

## 选择档位

单击该`Upscale`按钮可提升您图像的分辨率。这里我们选择将分辨率提升一倍。

![](./img/MJ_Upscale_Interface.jpg)
